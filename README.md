# 📞 PhoneOSINT — Advanced Phone Number Reconnaissance Tool

> **Author : CipherZeroo**  
> **GitHub : [CipherZeroo/phoneosint](https://github.com/CipherZeroo/phoneosint)**  
> **License: GPL-3.0-or-later**

PhoneOSINT is an open-source OSINT reconnaissance tool for international phone numbers. It parses, validates, and enriches phone number data using local carrier databases, Google dorking, and optional third-party APIs.

> ⚠️ **Authorized Use Only**

## Features

- Parse & validate any international phone number
- Carrier & line type detection
- Timezone resolution
- Approximate location with Google Maps link
- 70+ Google dork queries across 7 categories
- Batch scanning from file
- NumVerify & Twilio API integration (optional)
- JSON export
- Browser mode
